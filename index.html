PROJECTS

Country-list - Deploy Country-list - Code

Este projeto apresenta a possibilidade de exploração de países ao redor do mundo, fornecendo informações detalhadas sobre cada um deles. Ele foi desenvolvido com a finalidade de aprimorar conhecimentos nas tecnologias utilizadas, servindo como um projeto prático para aplicar os conceitos aprendidos. Next-JS, Javascript, HTML, CSS, Tailwind.

Space Invasion Game - Code

Este projeto de jogo foi desenvolvido como parte de um portfólio pessoal, demonstrando habilidades na criação de jogos com Python e a biblioteca Pygame Zero. Utilizando o PyCharm como IDE e o Anaconda como ambiente virtual para gerenciamento de dependências, o projeto busca explorar conceitos fundamentais de programação de jogos, como lógica de entrega, integração, interface e interatividade com o usuário.
